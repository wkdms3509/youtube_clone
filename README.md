# Youtude 반응형 모바일 (클론코딩)

## 개요

유튜브의 동영상 재생 화면 중 일부를 반응형 모바일 버전으로 직접 코딩을 해보며 그동안 공부한 문법을 일부 적용해보고 또 구글링과 관련 사이트의 정보를 통해 동영상 재생이나 태그의 여러가지 옵션을 새롭게 알게 되었고, css를 좀 더 다양하게 적용해 볼 수 있었다.
